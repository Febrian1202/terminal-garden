# 🚀 Hyprland Keybinds Cheatsheet (End-4 Theme)

> **Catatan:** `Super` = Tombol Windows/Meta.

## 💻 Aplikasi & Sistem

| Shortcut             | Fungsi                              |
| :------------------- | :---------------------------------- |
| `Super + Return`     | Buka Terminal (Kitty)               |
| `Super + B`          | Buka Zen Browser                    |
| `Super + E`          | Buka File Manager                   |
| `Super + C`          | Buka Code Editor (VS Code / Neovim) |
| `Super + I`          | Buka Pengaturan (Settings)          |
| `Ctrl + Shift + Esc` | Buka Task Manager                   |
| `Super + V`          | Buka Riwayat Clipboard              |
| `Super + .`          | Buka Picker Emoji                   |

## 🪟 Manajemen Jendela (Window)

| Shortcut        | Fungsi                                          |
| :-------------- | :---------------------------------------------- |
| `Super + Q`     | Tutup Jendela / Aplikasi (Kill)                 |
| `Super + Space` | Toggle Jendela Melayang / Tiling                |
| `Super + F`     | Fullscreen Total                                |
| `Super + D`     | Maximize (Bar tetap terlihat)                   |
| `Super + P`     | Pin Jendela (Selalu di atas)                    |
| `Super + \`     | Ubah Arah Belahan / Split (Horizontal/Vertikal) |

## ⌨️ Navigasi Ala Vim (HJKL)

| Shortcut                  | Fungsi                                       |
| :------------------------ | :------------------------------------------- |
| `Super + H/J/K/L`         | Pindah Fokus Jendela (Kiri/Bawah/Atas/Kanan) |
| `Super + Shift + H/J/K/L` | Geser Posisi Jendela (Kiri/Bawah/Atas/Kanan) |
| `Super + Ctrl + H/J/K/L`  | Ubah Ukuran (Resize) Jendela                 |

## 🖥️ Ruang Kerja (Workspace)

| Shortcut              | Fungsi                                           |
| :-------------------- | :----------------------------------------------- |
| `Super + 1-4`         | Pindah ke Workspace 1 s/d 4                      |
| `Super + Shift + 1-0` | Lempar aplikasi aktif ke Workspace 1 s/d 10      |
| `Super + S`           | Tampilkan/Sembunyikan Scratchpad (Ruang Rahasia) |
| `Super + Alt + S`     | Kirim aplikasi aktif ke Scratchpad               |

## 🎨 Kontrol Tema & Bar (Quickshell)

| Shortcut                      | Fungsi                           |
| :---------------------------- | :------------------------------- |
| `Super` (Tap) / `Super + Tab` | Buka App Launcher / Overview     |
| `Super + /`                   | Tampilkan menu Cheatsheet visual |
| `Super + A`                   | Buka Sidebar Kiri                |
| `Super + N`                   | Buka Sidebar Kanan (Notifikasi)  |
| `Super + Alt + J`             | Sembunyikan/Tampilkan Bar Atas   |
| `Ctrl + Super + T`            | Buka Pemilih Wallpaper           |
| `Ctrl + Super + Alt + T`      | Ganti Wallpaper Acak             |
| `Ctrl + Super + R`            | Restart/Muat Ulang Widget & Bar  |
| `Ctrl + Alt + Delete`         | Menu Power (Shutdown/Logout)     |

## ✂️ Tangkapan & Rekaman Layar (Utility)

| Shortcut                  | Fungsi                                      |
| :------------------------ | :------------------------------------------ |
| `Print Screen`            | Screenshot layar penuh (Salin ke Clipboard) |
| `Ctrl + Print Screen`     | Screenshot layar penuh (Simpan ke File)     |
| `Super + Shift + S`       | Screenshot area spesifik (Screen Snip)      |
| `Super + Shift + X`       | OCR (Ambil teks dari gambar di layar)       |
| `Super + Shift + C`       | Color Picker (Ambil kode warna Hex)         |
| `Super + Shift + A`       | Google Lens (Cari area layar di Google)     |
| `Super + Shift + R`       | Rekam area layar (Tanpa Suara)              |
| `Super + Shift + Alt + R` | Rekam seluruh layar (Dengan Suara)          |
| `Super + - / =`           | Zoom In / Zoom Out Layar                    |
